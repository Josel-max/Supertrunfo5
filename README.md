# 🃏 Supertrunfo5

Jogo de cartas Super Trunfo desenvolvido em linguagem C, como trabalho acadêmico do curso de Engenharia de Software.

## 🎮 Sobre o Jogo

Super Trunfo é um jogo de cartas onde os jogadores comparam atributos entre cartas e o maior valor vence a rodada. Este projeto implementa a lógica do jogo em C.

## 🛠️ Tecnologias Utilizadas

- Linguagem C

## ▶️ Como Executar

1. Compile o arquivo:
```bash
gcc Supertrunfo5.c -o Supertrunfo5
```
2. Execute o programa:
```bash
./Supertrunfo5
```

> ⚠️ **Atenção:** o jogo aceita apenas **letras minúsculas e sem espaços** nas entradas do usuário.

## 📚 O que aprendi

- Lógica de programação em C
- Estruturas de dados
- Desenvolvimento de jogos no terminal
